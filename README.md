# Xss Generator

XSS payload generator.

Generator payload XSS.


## Description

Creates Cross-Site Scripting payloads with various encoding and obfuscation techniques.

## Deskripsi

Membuat payload Cross-Site Scripting dengan berbagai teknik encoding dan obfuskasi.


## Installation

- Navigate to the project folder
- No dependencies required
- Open index.html in browser

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi
- Buka index.html di browser


## Usage

- Open index.html in browser
- Select payload type
- Choose encoding format
- Generate and copy payload

## Penggunaan

- Buka index.html di browser
- Pilih tipe payload
- Pilih format encoding
- Generate dan salin payload
